Alteração de texto

Temos as seguintes tags para fazer alterações em textos.

  • i Deixa o texto que estiver entre a tag em italico.
  • u Deixa o texto que estiver entre a tag sublinhado.
  • u Deixa o texto que estiver entre a tag sublinhado com pontos.
  • del Deixa o texto que estiver entre a tag com um traço ao meio da palavra
  • Strong Deixa o texto que estive entre a tag negrito.
  • small Deixa o texto que estiver entre a tag com fonte pequena.

Outros

Temos as seguintes tags

  • hr A tag hr cria uma linha na pagina geralmente usada para separar titulo de subtitulo.
  • sub A tag sub cria um texto na parte superior da linha
  • supA tag sub cria um texto na parte inferior da linha
  • font A tag não e suportado no HTML5 porem a tag era ultilizada no HTML4 para altera a cor, tamnho e tipo de fonte. Podemos fazer isso usando o CSS
